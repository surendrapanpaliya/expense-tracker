# Expense Tracker — Build Challenge

**Claude Code Lab 301 Capstone**

Your mission: build this entire app from `spec.md` in 60 minutes using Claude Code.

## What You Get

- `spec.md` — the full product specification
- That's it. You build everything else.

## What You Deliver

A working REST API with:
- All endpoints from the spec
- Business rule enforcement (budget limits, status transitions)
- A passing test suite
- A CLAUDE.md documenting your project
- Hooks enforcing quality gates

## Rules

- You choose the tech stack (Java, Node.js, or Python)
- Claude Code does the heavy lifting — your job is to guide it
- Use everything you've learned: plan mode, CLAUDE.md, hooks, worktrees, headless validation
